Лабораторная работа №3 — Конфигурационное управление (Ansible)
Цель работы
Изучить Ansible, научиться развёртывать роли и создавать собственные модули.

Состав проекта
play.yml — основной плейбук
roles/nginx — роль установки и настройки NGINX
modules/my_info.py — пользовательский модуль информации
modules/my_facts.py — пользовательский модуль фактов
Выполненные шаги
Настроен управляющий узел (ПК) и управляемый узел (ноутбук).
Проверено SSH-соединение.
Создана роль NGINX и применена на управляемом узле.
Разработаны собственные модули и протестированы.
Результаты
Ansible успешно выполняет задачи, NGINX установлен, модули работают корректно.
